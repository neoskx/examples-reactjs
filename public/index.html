Official Tutorial

This is the official Tutorial provided by ReactJS. You can get more information from tutorial

After this tutorial you should know following knowledge

Custom React Class/Component names must begin with an uppercase letter.

Create React Component

// CommentBox is Custom React Class/Component name
let CommentBox = React.createClass({
  render(){
    return (
      <div className="commentBox">
        Hello, world! I am a CommentBox.
      </div>
    );
  }
});

Using props

// Use

this.props.author
this.props.children

Dangerously set inner html

You can use lodash to encode or decode HTML Entity. _.escape_.unescape

// Get inner html
function getInnerHTML(){
  return {__html: "<h1>Inner HTML</h1>"}
}
// Use
